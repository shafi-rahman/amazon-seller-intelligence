# ASIP SEO Agent — OpenClaw Skill

Monitors ASIP for newly generated SEO campaigns and sends you a WhatsApp/Telegram
notification when posts are ready to review.

## Setup

### 1. Install dependencies

```bash
cd openclaw-skill
npm install
```

### 2. Configure

Edit `seo-agent.js` or set environment variables:

```bash
export ASIP_URL=http://localhost:7801
export SEO_WEBHOOK_TOKEN=<value from ASIP .env SEO_WEBHOOK_TOKEN>
export ASIP_WORKSPACE_ID=1
```

### 3. Run

**Standalone (console notifications):**
```bash
node seo-agent.js
```

**As an OpenClaw Skill:**
1. Install OpenClaw: `curl -fsSL https://openclaw.ai/install.sh | bash`
2. Copy `seo-agent.js` to your OpenClaw skills folder
3. Tell OpenClaw: *"Run the ASIP SEO agent every 5 minutes and notify me on WhatsApp"*

## What It Does

```
Every 5 minutes:
  → Calls ASIP API for campaigns in 'awaiting_approval' status
  → For each new campaign: sends WhatsApp/Telegram message
  
Message:
  🎯 SEO Posts Ready for Review!
  📦 Product: Blue Ceramic Coffee Mug 350ml...
  ✨ 4 posts generated by NVIDIA Nemotron
     • 📸 Instagram caption + hashtags
     • 📘 Facebook story post  
     • 💼 LinkedIn professional post
     • 🔍 Google Business post
  
  👆 Tap to Review & Approve:
  http://localhost:7801/seo/campaigns/5
```

## Approval Flow

1. Receive WhatsApp notification
2. Click link → opens ASIP in browser
3. Review each post (Instagram, Facebook, LinkedIn, Google)
4. Edit caption if needed → click Approve
5. Copy approved post → paste in social media app on your phone
